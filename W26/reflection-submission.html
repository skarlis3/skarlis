Submission

Final Reflection — Due: Monday, May 5

How you submit your Final Reflection depends on the format you chose.

PowerPoint or Google Slides

Upload your file to the Final Reflection assignment in Canvas as a .pptx file. If you used Google Slides, download it first: File > Download > Microsoft PowerPoint (.pptx).

Google Site or Other Website

Submit the published link to your site in the Final Reflection assignment in Canvas. Make sure the link goes to the published version of your site, not the editing view.

Make sure you have the right link. The published URL for Google Sites looks like: sites.google.com/view/your-site-name

If the URL contains /edit or /d/, you are looking at the editing link, not the published site. I will not be able to view your site with the editing link.

Before You Submit

  • All six sections are included and substantive
  • Sections 1–4 include portfolio artifacts
  • Visual design is intentional and readable
  • If using a website, the link works and the site is publicly viewable
  • If using slides, the file is saved as .pptx
